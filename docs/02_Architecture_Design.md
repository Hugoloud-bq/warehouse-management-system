# Архитектура системы складского учёта

## 1. Общая архитектура
Система использует трёхуровневую архитектуру:
1. **Уровень данных (Data Layer):** Microsoft SQL Server 2019
2. **Бизнес-логика (Business Layer):** ASP.NET Core Web API
3. **Презентационный уровень (Presentation Layer):** React SPA

## 2. Компоненты системы
- **База данных:** Microsoft SQL Server 2019
- **Backend:** ASP.NET Core 6.0 Web API
- **Frontend:** React 18 + TypeScript
- **Аутентификация:** JWT токены
- **Стили:** Material-UI компоненты
