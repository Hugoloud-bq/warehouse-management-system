# Техническое задание
## на разработку автоматизированной системы складского учёта

### 1. Общие сведения
- **Наименование системы:** Автоматизированная система управления складом (АСУ Склад)
- **Условное обозначение:** АСУ-СКЛ-2025
- **Заказчик:** Учебный проект
- **Разработчик:** Чуляков Семён Игоревич
- **Основание для разработки:** Курсовой проект по дисциплине "Проектирование ИС"
- **Сроки разработки:** 21.11.2025 - 25.12.2025

### 2. Назначение и цели создания
**Цель:** Автоматизация процессов складского учёта на предприятии.

**Задачи:**
- Ликвидация "бумажного" учёта
- Сокращение времени на поиск товаров на 60%
- Уменьшение ошибок при инвентаризации на 90%
- Повышение точности учёта до 99.5%

### 3. Требования к системе

#### 3.1. Функциональные требования
**Модуль учёта товаров:**
- Ведение номенклатуры с учётом характеристик (вес, объём, срок годности)
- Привязка товаров к конкретным местам хранения
- Управление минимальными/максимальными остатками

**Модуль документооборота:**
- Формирование приходных накладных
- Создание расходных документов
- Регистрация внутренних перемещений
- Проведение инвентаризаций

**Модуль отчётности:**
- Отчёт об остатках товаров
- Оборотно-сальдовая ведомость
- Отчёт по движениям товаров за период
- Отчёт по срокам годности

#### 3.2. Нефункциональные требования
- **Производительность:** Обработка до 1000 операций в час
- **Надёжность:** Время безотказной работы 99.9%
- **Безопасность:** Разграничение прав доступа по ролям
- **Масштабируемость:** Поддержка до 100 рабочих мест

#### 3.3. Требования к интерфейсу
- Интуитивно понятный интерфейс
- Поддержка русского языка
- Адаптивность для разных разрешений экрана
- Минимальное количество кликов для выполнения операций
