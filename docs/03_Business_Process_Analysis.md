# Анализ бизнес-процессов

## 1. Текущее состояние (AS-IS)

### 1.1. Управление товарами
- Учёт ведётся в бумажных журналах
- Отсутствие единой системы учёта
- Частые ошибки при подсчёте остатков
- Длительный поиск товаров на складе

### 1.2. Документооборот
- Ручное заполнение накладных
- Дублирование информации
- Отсутствие контроля за движением товаров
- Сложность формирования отчётов

### 1.3. Инвентаризация
- Проводится 1 раз в квартал
- Занимает 3-5 рабочих дней
- Высокий процент ошибок (до 15%)
- Длительная сверка результатов

## 2. Целевое состояние (TO-BE)

### 2.1. Управление товарами
- Автоматический учёт всех операций
- Мгновенный поиск товаров по ячейкам
- Контроль минимальных/максимальных остатков
- Автоматическое формирование заказов

### 2.2. Документооборот
- Электронное оформление документов
- Автоматический расчёт сумм
- Контроль цепочки движения товаров
- Мгновенное формирование отчётов

### 2.3. Инвентаризация
- Возможность выборочной инвентаризации
- Сокращение времени до 1 дня
- Снижение ошибок до 1%
- Автоматическая сверка результатов

## 3. Основные бизнес-процессы

### 3.1. Приёмка товара
1. Создание приходной накладной
2. Проверка соответствия товара документам
3. Размещение товара на складе
4. Подтверждение приёмки

### 3.2. Отпуск товара
1. Создание расходной накладной
2. Резервирование товара
3. Отбор товара со склада
4. Подтверждение отпуска

### 3.3. Внутренние перемещения
1. Создание заявки на перемещение
2. Подтверждение менеджером
3. Физическое перемещение
4. Обновление данных в системе

### 3.4. Инвентаризация
1. Создание плана инвентаризации
2. Проведение подсчёта
3. Внесение результатов в систему
4. Формирование акта расхождений

## 4. Ожидаемые результаты

### 4.1. Экономические показатели
- Сокращение времени поиска товаров: 60%
- Уменьшение ошибок при инвентаризации: 90%
- Повышение точности учёта: 99.5%
- Сокращение персонала: 20%

### 4.2. Операционные показатели
- Увеличение скорости обработки заказов: 50%
- Сокращение времени инвентаризации: 70%
- Увеличение оборачиваемости склада: 30%
- Снижение потерь от пересорта: 95%
